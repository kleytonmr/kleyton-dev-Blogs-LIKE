Este é outro post de exemplo, desta vez nas categorias de **ciencia** e **aprendizado**.

## Estrutura de um Post

Cada post deve ter um cabeçalho YAML (front matter) no início:

```yaml
---
layout: post
title: "Título do Post"
date: 2024-01-20 14:30:00 -0300
categories: [categoria1, categoria2]
tags: [tag1, tag2, tag3]
excerpt: "Breve descrição do post"
---
```

## Campos Importantes

- **layout**: Sempre use `post`
- **title**: Título do seu post
- **date**: Data e hora no formato YYYY-MM-DD HH:MM:SS -0300
- **categories**: Array de categorias (tópicos) - use colchetes []
- **tags**: Array de tags - use colchetes []
- **excerpt**: Resumo que aparece na listagem (opcional)

## Organização por Tópicos

Você pode criar quantas categorias quiser. Alguns exemplos:
- tecnologia
- programacao
- ciencia
- aprendizado
- filosofia
- viagens
- receitas

Basta adicionar as categorias no campo `categories` do front matter!
